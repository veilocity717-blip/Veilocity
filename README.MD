
# Veilocity — Multilingual Archive  
(Online Mirror + Patreon Frontend + Offline Archive)

This repository is a multilingual archive of my social media writings, designed for preservation, accessibility, and redistribution.

This is not entertainment — it’s remembrance.

---

## Architecture Overview

This project operates across three layers:

### 1. GitHub (This Repository)
- archive structure  
- reference layer  
- mirror of content  

This is **not the primary interface**.

---

### 2. Patreon (Online Frontend)
Primary online access point:

https://www.patreon.com/c/stephenchristophercooke/collections  

- structured presentation  
- curated access to content  
- easiest way to explore online  

---

### 3. Offline Archive (Primary System)

Distributed via Google Drive and MEGA:

- complete archive  
- full asset integrity  
- portable frontend included  
- no dependency on platforms  

This is the **intended experience**.

---

## Purpose

This archive preserves a body of work centered on:

- discernment  
- personal sovereignty  
- spiritual clarity  
- perception and awareness  
- autonomy and internal alignment  

Each entry is compiled across multiple languages to ensure accessibility beyond borders, platforms, and algorithms.

This is a living archive — structured, intentional, and designed to persist independently of centralized systems.

## Offline Usage (Recommended)

1. Download a volume  
2. Extract files  
3. Open:

Arkchive Multilingual Frontend/index.html

No installation required  
No internet required

## Portable Frontend

The included frontend:

- auto-detects archive folders  
- displays by country  
- dynamically loads flags  
- links directly into each archive  
- provides search across all archives  

### Notes

- built for **offline/local use**
- ignores:
  - version suffixes (`v015`)
  - files containing `court` in filenames
