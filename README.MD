Veilocity — Multilingual Archive

(Online Mirror + Patreon Frontend + Offline Archive)

This repository is a multilingual archive of my social media writings, designed for preservation, accessibility, and redistribution.

This is not entertainment — it’s remembrance.

Architecture Overview

This project operates across three layers:

1. GitHub (This Repository)
archive structure
reference layer
mirror of content

This is not the primary interface.

2. Patreon (Online Frontend)

easiest way to explore online
3. Offline Archive (Primary System)

Distributed via Google Drive and MEGA:

complete archive
full asset integrity
portable frontend included
no dependency on platforms

This is the intended experience.
